# Sprint Report — Sprint 5

## Sprint Goal

The goal of Sprint 5 was to set up the achitecture and implement localization.

The sprint focused on moving from static strings into localization calls to support the existing languages and other languages to come.

- Implemented language selector in the UI
- Added 2 non-Latin languages (Arabic and Japanese)
